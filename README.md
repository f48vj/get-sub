# get-sub

get subtitles from shooter.cn

## Install

### npm

```bash
npm install get-sub -g
```
### yarn

```
yarn global get-sub
```

## Usage

### basic

```bash
cd <dir>
get-sub
```

or

```bash
get-sub <dir>
```

### with options

#### get *english* subtitles(default is *chinese*)

```bash
get-sub -l eng [dir]
```

### set exts of video files(default is mkv)

```bash
get-sub -e mkv,mp4 [dir]
```
